Matix - Object

Presentation

An Object is something used in a scene.
Each objects has a transformation, and can have (or not) a graphic apparence and a physic part.

Transform Object

Each object has a transformation. A transformation represents a position, a rotation and a scale.
The transformation can also includes a parent, which will apply its transformation to its son.

Position handle

The position handling is the easiest handling in the transform.
Each object has a local position, in its parents.
All the transformations to the object start from an anchored position, except the ones coming from the parent.
To not interfere with the local position, this kind of position are located in a position offset (one for parent and one for anchor).
The object can add all that to have its absolute position in the scene.
The parent absolute position is also added into the calcul of the object absolute position.
