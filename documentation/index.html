Matix

Presentation

Matix is a little OpenGL library developed by a 17yo (february 2024) french guy.
It's created for fun and to be shared. Because of the tiny experience of its developer, it mays not be perfect everywhere.

The goal with this library is to allow 3D display and basics 2D display easily.
It uses C++ and OpenGL to work. To use OpenGL with C++, GLFW, ModernGL and Glad are used. Some other libraries like stb_image are also used.

How to start ?

Requirement

To use this library, you will need the GLFW, ModernGL, Glad and stb_image libraries.
A minimum level of C++ and OpenGL is also needed.
